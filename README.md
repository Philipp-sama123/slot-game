### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Philipp-sama123/slot-game.git
   cd slot-game
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser and play:**
   - Navigate to `http://localhost:8080/` and enjoy
